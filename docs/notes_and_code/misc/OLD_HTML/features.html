Step 1: Enter Locations

Enter your starting and ending location in the top right corner box under starting location. Try to be as specific as possible.

Step 2: Choose Vehicle

There is a drop-down menu of possible vehicles. There is a limited number of vehicles, so if you cannot find your vehicle, try to find a vehicle that is similar. Values for mass, drag coefficient and frontal surface area will be updated when you change the vehicle selection.

Step 3: Press Navigate!

Once the program has finished calculating the route, a popup box will appear with the directions of navigation.
